# タスクアプリ

## 概要
目標達成のためのタスクアプリです。<br>
タスク達成率のランキング表示やグループへの参加などでやりたくなる、やらざるを得ない状況を作り出し、個人の目標を達成します。


## 制作背景
タスクを管理できるアプリは多くありますが、個人で完結するものがほとんどです。<br>
個人のモチベーションアップはもちろん、やる気がないときでも個人の意思とは無関係にやらざるを得ない環境にして、目標達成を少しでも手助けでき、その人の人生が少しでも良い方向に向けばと思い作成しました。


## URL
https://task-achievement.herokuapp.com/<br>
ログイン画面の「ゲストユーザーとしてログイン」をクリックすると、会員登録をせずにログインできます。


## 機能一覧

#### ユーザー機能
- 新規登録、ログイン、ログアウト
- プロフィール表示、更新


#### タスク機能
- タスクの一覧、作成、更新、削除
- タスクの達成率の表示
　

#### グループ機能
- グループの一覧、検索、作成、詳細、参加
- グループに参加することで、グループ全体、グループメンバーのタスク達成率の表示


#### ランキング機能
- タスク達成率によって、個人(当日、1週間)、グループ(当日、1週間)ごとにランキングの表示


#### カレンダー機能
- 各日付のタスク達成率の表示
- 過去のタスクの修正


#### グラフ機能
- タスク達成率をグラフで表示
